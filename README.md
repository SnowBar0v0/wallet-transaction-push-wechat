# wallet-transaction-push-wechat

链上钱包交易推送-依附AlertDog监控狗

## 功能

1-实时推送交易信息至微信（短时间大量信息推送至微信可能触发风控）"/启动"或"/关闭"进行开关此功能

2-在TG或微信中输入如"/交易 1-12" 则汇总当日1点至12点的交易数据，输入"/转账 1-2"同理

3-在2的基础上每晚24点时会进行一次汇总

4-每隔8小时会自动进行一次汇总交易和转账的数据

## 注意事项

需要把监控bot拉入频道中才可跑通脚本，普通群组不行（群组中获取不到信息的字段）

理论上任意监控bot都可以拉频道中运行此脚本，不过汇总功能则会失效
